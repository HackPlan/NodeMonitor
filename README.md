# NodeMonitor

服务器上的状态监视器，支持查看：

* 服务器操作系统
* 服务器当前时间
* 服务器已运行时间
* CPU型号
* CPU使用状况
* 硬盘使用状况
* 内存使用状况(包括物理内存/Cache/真实内存/SWAP分区)
* 系统平均负载
* 用户进程状况
* 所有用户进程状况
