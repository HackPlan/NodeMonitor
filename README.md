# NodeMonitor

Status Monitor on the server, supports viewing: 

* OS
* Server time 
* Uptime 
* CPU model 
* CPU usage 
* Hard disk usage 
* Memory usage (including physical memory / Cache / real memory / SWAP partition) 
* System load average 
* User process
* All user process
